      Kebkab Events – Event Organizer Website

-Kebkab Events is a responsive, multi-page website designed to showcase Orthodox event planning services in Addis Ababa, Ethiopia. 
   The project focuses on clean design, usability, and professional presentation.

    *Features

  -Responsive navigation with mobile menu

  -Hero sections with call-to-action buttons

  -Statistics section highlighting experience

  -Multi-page structure (Home, About, Events, Gallery, Contact)

  -Contact and social media integration

   *Technologies Used

 -HTML – Page structure and semantics

 -CSS – Layout, styling, and responsiveness

 -JavaScript – Navigation interactivity

-GitHub Pages – Hosting and deployment

* How to Run

-Download or clone the project.

-Open index.html in a browser.


* Project Structure

-index.html – Home page

-about.html, events.html, gallery.html, contact.html

-style.css – Website styling

-script.js – Interactive behavior

* Purpose

-This project was developed as a Web Programming course assignment to demonstrate responsive design, semantic HTML, CSS styling, and basic JavaScript interactivity.
